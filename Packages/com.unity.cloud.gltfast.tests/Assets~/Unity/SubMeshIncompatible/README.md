# SubMeshIncompatible

## Description

Contains meshes with more than one primitive of different vertex buffer layout (one is missing texture coordinates). Those are supposed to be imported into two meshes with one sub-meshes each, as Unity does not support different vertex buffer layouts per sub-mesh.

This is a variant of the *SubMesh* model.
